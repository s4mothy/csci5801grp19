PPALMS Version 0.1

Required Software:
Gradle 7.3
JDK 17.0.5

Run:
1. In csci5801grp19 directory open a command line and run ./gradlew run
2. Once prompted to enter path for specification file, enter the absolute path to an input file (ez_quiz.txt)
3. The problems generated will be seen following the title within the terminal after running in order from the question, choices, and then solution.
	3a. Additionally, the problems generated will be found in ..\csci5801grp19\app\src\main\resources\stored_probs. However, these are in hex and bascially unreadable.
